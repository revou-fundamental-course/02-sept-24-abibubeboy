LUAS SEGITIGA

Rumus luas segitiga yaitu:

L = 1/2 x a x t

Di mana:
L = Luas
a = Alas
t = Tinggi

KELILING SEGITIGA

Rumus keliling segitiga yaitu:

K = s1 + s2 + s3

Di mana:
K = Keliling
s1 = A
s2 = B
s3 = C
